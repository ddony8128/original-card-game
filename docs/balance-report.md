## A. 매치업 승률 / 평균 턴수

매치업당 100판. "A승" = 표 왼쪽(스테이지 AI) 승률.

| 매치업 | A승률(seed50k/200k) | 무 | 평균 턴 |
|---|---|---|---|
| 1 무투법사 vs 기본덱 | **24% / 33%** | 0 | ~8.7 |
| 2 게임개같이하네 vs 기본덱 | 52% / 39% | 7 | ~27 |
| 3 노루약해요 vs 기본덱 | **86% / 86%** | 0 | ~15 |
| 1 vs 2 | 42% / 48% | 0 | ~13 |
| 1 vs 3 | 19% / 22% | 0 | ~11 |
| 2 vs 3 | 8% / 11% | **무 60~66** | ~62 |

게임 길이(900판 합산): 원턴킬(≤2턴) 0%, 3~10턴 ~36%, 11~40턴 ~52%, 41~79턴 ~3%, **80턴 cap 무승부 ~8.6%**.

### 해석 (이상치)
1. **난이도 곡선 양극화.** "1단계" 무투법사가 기본덱 상대 24~33%로 제일 약하고, "3단계" 노루약해요가 86%로 압도. 1단계=거의 무료 승리, 3단계=거의 무료 패배 수준.
2. **2 vs 3 교착(무 60%+).** 둘 다 컨트롤/카이팅(밀·힐·드로우)이라 80턴 cap까지 결판 안 남. 피니셔 부족.
3. **어그로(무투)가 컨트롤(노루)에 구조적 열세**(1 vs 3 ≈ 20%). 노루의 힐(운기조식·지력흡수) + 운석 피니셔가 어그로 클럭을 압도.

### 하네스 한계 (중요)
- **하드 스테이지(4/5/6)의 `aiHp=30` 이 selfPlay 에 미반영.** `aiHp` 는 `soloGameManager.ts` 에만 적용되고 `selfPlay` 의 초기 상태 생성엔 안 들어가, 4/5/6 결과가 1/2/3 과 완전히 동일하게 나옴. 실제 게임은 AI HP +10 만큼 더 어려움. (→ 시뮬 충실도를 높이려면 selfPlay 초기상태에 aiHp 반영 필요)
